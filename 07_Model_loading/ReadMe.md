### 提几个需要注意的地方
* 文件内容直接读就可以，不用二进制读
* 将文件中一长串字符输出到一个`string`中，遇到空格会结束
* 载入的`DDS`文件，则需要把`uv`坐标的`y`坐标变成相反值，即`uv.y=-uv.y`
* 对于` d/d/d`这样的输入，如果是用`fscanf`则可以用格式输入，`fscanf`的参数是`FILE*`，但是用`fstream`输入流则没有这个功能，所以直接用文件输入流，中间的`/`用一个临时`char`型变量接收就可以了
* 最后还有一点需要切记的是一定要调用`file.close()`，之前没关文件会导致调试多次的时候内容被覆盖，重复等问题。

#### 文件格式
- #是注释标记，就像C++中的//
- `usemtl`和`mtlib`描述了模型的外观。本课用不到。
- v代表顶点
- vt代表顶点的纹理坐标
- vn代表顶点的法向
- f代表面

#### v vt vn都很好理解。f比较麻烦。例如`f 8/11/7 7/12/7 6/10/7`：

- 8/11/7描述了三角形的第一个顶点
- 7/12/7描述了三角形的第二个顶点
- 6/10/7描述了三角形的第三个顶点
- 对于第一个顶点，8指向要用的顶点。此例中是-1.000000 1.000000 \-1.000000（索引从1开始，和C++中从0开始不同）
- 11指向要用的纹理坐标。此例中是0.748355 0.998230。
- 7指向要用的法向。此例中是0.000000 1.000000 -0.000000。
